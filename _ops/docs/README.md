# مرکز عملیات پروژه (OPS)

این شاخه، خروجی کامل ممیزی، نقشه‌راه فازی، اجرای خودکار، لاگ‌ها و گزارش نهایی را نگه می‌دارد.

## فهرست
- `audit/discovery.md`: کشف ساختار پروژه و اجزای کلیدی
- `audit/stack_versions.md`: پشته فناوری و نسخه‌ها
- `audit/architecture.md`: معماری، جریان درخواست و احراز هویت
- `audit/dependencies.md`: تحلیل وابستگی‌ها و ریسک زنجیره تامین
- `audit/security.md`: خط مبنای امنیت، OWASP، شکاف‌ها و اصلاحات
- `audit/performance.md`: بررسی کارایی و بهینه‌سازی‌های سریع
- `audit/devops_deploy.md`: آمادگی DevOps/استقرار و Docker
- `audit/testing_quality.md`: وضعیت تست و کیفیت خودکارسازی
- `audit/code_quality.md`: کیفیت کد، نگهداشت‌پذیری و استانداردها
- `audit/evidence.md`: شواهد `path:line`
- `../roadmap/roadmap_master.md`: نقشه‌راه فازی بدون تاریخ
- `../roadmap/roadmap_data.json`: داده ساخت‌یافته تسک‌ها برای داشبورد
- `../dashboard/index.html`: داشبورد آفلاین وضعیت اجرا
- `../logs/actions.log`: لاگ اجرای فرمان‌ها
- `../logs/errors.log`: لاگ خطاها و موارد Blocked
- `../reports/final_report.md`: گزارش نهایی آمادگی انتشار
