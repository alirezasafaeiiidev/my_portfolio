# asdev-portfolio

Next.js (App Router) portfolio application with an admin/API surface, optimized for production deployment on a VPS.

## Quickstart

- Install: `pnpm install`
- Dev: `pnpm dev`
- Build: `pnpm build`
- Start (prod): `pnpm start`
- Verify: `pnpm run verify`

> Runtime requirements are defined in `package.json` (`node` 20–22, `pnpm` 9–10).
