# ID: P2-01
## Title
Export/import history & settings (local-first portability)

## Context
- Users may want to migrate to a new device while keeping preferences.

## Steps
1. Add “Export data” to download a JSON file of settings/history metadata.
2. Add “Import data” with validation + merge strategy.
3. Add UX for partial import (settings only vs full).

## Acceptance criteria
- Exported JSON can be imported back to reproduce settings/history.
- Invalid imports show clear Persian errors and do not corrupt existing data.
