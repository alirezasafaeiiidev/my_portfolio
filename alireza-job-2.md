# ASDEV Portfolio - Alireza Job 2 (Checklist)

این فایل فقط مواردی است که برای 10/10 شدن پروژه، باید از شما دریافت شود. هر چیزی که بدون اطلاعات شما ممکن بوده انجام شده/در حال انجام است.

## Must-Provide (High Impact)
- [ ] CTA اصلی سایت را انتخاب کن (فقط یکی):
  - `درخواست ارزیابی` یا `درخواست پروژه` یا `رزرو جلسه`
- [ ] 3 تا 6 خدمت اصلی که واقعا می‌فروشی (عنوان + یک جمله خروجی قابل تحویل برای هرکدام)
- [ ] بازه قیمت یا “شروع از …” برای هر خدمت (اگر نمی‌خواهی عمومی باشد: مشخص کن فقط در فرم/پیگیری گفته شود)
- [ ] قوانین نمایش شماره تلفن:
  - [ ] شماره در سایت عمومی نمایش داده شود (الان نمایش داده می‌شود)
  - [ ] یا فقط داخل فرم باشد و از UI حذف شود
- [ ] زبان پیش‌فرض سایت را تایید کن: `فارسی` یا `انگلیسی` (الان: فارسی)

## Proof / Portfolio (Conversion)
- [ ] لینک Live و نقش دقیق شما برای هر کیس‌استادی:
  - PersianToolbox: نقش + 2 تا 3 عدد “نتیجه قابل اندازه‌گیری” (مثلا LCP/SEO/ریسک/MTTR/Conversion)
  - Portfolio: همین‌ها (هدف/نتیجه/Tradeoff)
- [ ] اگر Repo عمومی برای PersianToolbox یا Portfolio هست: لینک بده (اختیاری اما اعتمادساز)
- [ ] لیست “پروژه‌های در حال توسعه” را تایید کن (نام + وضعیت + آیا نمایش عمومی بشود؟):
  - asdev-automation-hub
