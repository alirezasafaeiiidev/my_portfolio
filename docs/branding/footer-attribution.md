# Footer Attribution Contract

## Required Footer Line

Add one persistent attribution line in primary footer:

- EN: `Built by Alireza Safaei (ASDEV)`
- FA: `توسعه و اجرا توسط علیرضا صفایی (ASDEV)`

## Link Policy

- Attribution must link to canonical personal site when domain is available.
- Link target must use `rel="noopener noreferrer"` for external navigation.

## Placement Rule

- Attribution is visible on all public pages.
- Footer attribution must not be hidden behind drawers/modals.

## Verification

- Desktop and mobile screenshots prove visibility.
- Footer line remains visible after release build, not only dev mode.
