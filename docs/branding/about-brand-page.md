# About Brand Page Contract

## Minimum Route

- Public route required: `/about-brand` or `/brand`
- Route must be included in sitemap (if indexable) and internal navigation.

## Required Sections

- Brand owner identity (person + role)
- Brand mission and operating principles
- Relationship between product brand and master brand (ASDEV)
- Contact/action path for consulting/commercial inquiries

## Required Metadata

- Unique title + description aligned with brand positioning
- Canonical URL for the brand page
- JSON-LD: `Person` or `Organization` reference aligned with site entities

## Acceptance

- Page is reachable in max 2 clicks from homepage.
- Footer includes direct link to this page.
- Copy is consistent with hero positioning and service page messaging.
